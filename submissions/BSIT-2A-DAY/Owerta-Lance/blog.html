OOP

Object-Oriented Programming (OOP) is a programming paradigm that organizes code around objects, which are data structures that contain both data (attributes) and methods (functions) that operate on that data. It is a way of structuring programs by modeling real-world entities as objects, making code more modular, reusable, and easier to manage, especially for large and complex software.
